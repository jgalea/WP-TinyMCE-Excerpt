== Description ==

Adds the TinyMCE rich text editor to the Excerpt box in Add Post and Edit Post screens.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload \`wp-tinymce-excerpt\` to the \`/wp-content/plugins/\` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
